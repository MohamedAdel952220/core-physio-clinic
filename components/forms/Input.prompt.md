**Input** — labeled form field for the booking form (name, phone, age, date). Calm teal focus ring, RTL-aware.

```jsx
<Input label="Full Name" placeholder="e.g. Sara Mahmoud" required />
<Input label="Preferred Date" type="date" required />
```

Pass `type="date"` for the date picker, `icon` for a leading glyph, `dir="rtl"` for Arabic, `hint` for helper text.
