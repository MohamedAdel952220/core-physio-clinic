**Badge** — compact trust/status chip for hero trust badges, ratings, and tags.

```jsx
<Badge tone="gold" icon={<Star/>}>4.9 Google Rating</Badge>
<Badge tone="teal" dot>Open now</Badge>
```

Tones: `teal`, `gold` (ratings), `neutral`, `soft` (on imagery/glass), `whatsapp`. Pass `dot` for a status dot.
